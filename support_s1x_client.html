Below are guides to help you with any client issues you may have with S1x.

Experiencing errors? View our error page HERE.

S1x CLIENT INSTALL GUIDE

If You Do Not Own The Game On Steam

Requirements:

  • A Torrent Client. (We recommend qBittorrent)
  • The S1x Full Game torrent (NOT CURRENTLY AVAILABLE)
  • The S1x Client Files (NOT CURRENTLY AVAILABLE)

Getting Started:

  1. Download the torrent which contains the full game including the DLC content.
  2. Move the folder to a location like C:\Games\S1x.
  3. Download S1x.exe and place it in the game directory.
  4. To start the game, double-click S1x.exe.

To change your name, open the in-game console (press the ~ key) and type /name yourname. (May not be available)

To unlock all, open the in-game console (press the ~ key) and type /unlockstats. (May not be available)

S1x STEAM INSTALL GUIDE

If You Do Own The Game On Steam, And want DLC/S1x.

Requirements:

  • A Torrent Client. (We recommend qBittorrent)
  • The S1x Client Files (NOT CURRENTLY AVAILABLE)
  • The S1x DLC Files (NOT CURRENTLY AVAILABLE)

Getting Started:

  1. Download the torrent which contains the DLC content.
  2. Move the completed download into your Steam Advanced Warfare Folder.
  3. Download and place S1x.exe in the game folder.
  4. To start the game, double-click S1x.exe.
